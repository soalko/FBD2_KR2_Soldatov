## Функциональность

### Аутентификация и авторизация
- Регистрация новых пользователей
- Вход в систему с получением JWT токенов
- Автоматическое обновление access-токена через refresh-токен
- Хранение токенов в `localStorage`
- Защищенные маршруты на клиенте и сервере

### Управление товарами
- Просмотр списка товаров с фильтрацией по категориям
- Поиск товаров по названию и описанию
- Просмотр детальной информации о товаре
- Создание новых товаров (для продавцов и админов)
- Редактирование товаров (для продавцов и админов)
- Удаление товаров (только для админов)

### Управление пользователями (только для админов)
- Просмотр списка всех пользователей
- Изменение ролей пользователей
- Блокировка пользователей

---

## Роли и права доступа

| Роль | Доступные действия |
|------|-------------------|
| **Гость** (не аутентифицирован) | Регистрация, вход в систему |
| **Пользователь** (`user`) | Просмотр товаров |
| **Продавец** (`seller`) | Просмотр, создание, редактирование товаров |
| **Администратор** (`admin`) | Полный доступ: управление товарами и пользователями, изменение ролей |
